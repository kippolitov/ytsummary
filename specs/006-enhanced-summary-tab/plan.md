# Implementation Plan: Enhanced Summary Tab — Knowledge Surface

**Branch**: `006-enhanced-summary-tab` | **Date**: 2026-06-09 | **Spec**: [spec.md](spec.md)

**Input**: Feature specification from `specs/006-enhanced-summary-tab/spec.md`

## Summary

Transform the summary tab from a single prose paragraph + shallow topic list into a TL;DR bullet list (3–7 items) paired with enriched topic entries (2–5 sentence explanations with insight). The change touches the Azure Function response model, the OpenAI prompt, and two React components. No new infrastructure or dependencies are introduced.

## Technical Context

**Language/Version**: TypeScript 5.5 (extension + functions)

**Primary Dependencies**:
- Extension: React 18, WXT 0.19, Tailwind CSS 3.4, Vitest 1.6, @testing-library/react 16, Playwright 1.44
- Functions: Azure Functions v4, Azure OpenAI SDK, Vitest 1.x

**Storage**: Session-scoped in-memory cache (`extension/services/sessionCache.ts`) — `AnalysisResult` keyed by `videoId`; no persistent storage

**Testing**: Vitest (unit, both packages) + Playwright (e2e, extension)

**Target Platform**: Chrome MV3 side panel (Chrome extension)

**Performance Goals**: Analysis result delivered ≤ 30 s for 10-minute video (Constitution QG-4); loading indicator within 3 s (SC-005)

**Constraints**: `max_tokens: 2000` for OpenAI completion call (sufficient — see [research.md Decision 5](research.md)); no new npm dependencies

**Scale/Scope**: Single-user browser extension; one active video analysis at a time
